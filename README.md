# AstroMe
### Tematicka socialna siet na zakazku  
- Podpora upravy profilu (profilova fotka, bio)
- pridavanie prispevkov a fotiek
- followovanie pouzivatelov a feed  
- likeovanie prispevkov a komentare  
- ukladanie denneho reportu     
- chatovanie  

**Technologie**
- Aplikacia postavana na **Reacte** (FE) a **Firebase** (BE)
- implementovane rozne API 3 stran
- Cloudinary pouzite na ukladanie obrazkov
